INTRODUCTION

Welcome! This asset pack contains a set of 50+ pixel art sprites you can use to create a graveyard-themed level.

Each group of tombstones, coffins, and mausoleums contains a set of 5 at each size of small, medium, and large. The other sprites in this pack include flowers, gates, statues, candles, and ground tiles.

NOTE: Updates to this asset pack are currently on hold. A roadmap is currently in-work to determine what other assets are best to include in this pack.
