# PAINT PROGRAM SHOPPING — Master Plan (41 zones merged)

Merged from all 41 zone shopping lists. Section 1 orders the generation queue by total demand. Section 2 is the deduped descriptor library to generate — tags name the consuming zones. Section 3 preserves every zone's story-site list verbatim; the paint phase executes those compositions once the library completes.
